Power

  • Researchers test the following hypotheses to assess whether the difference in treatment \(\mu\) is significantly different

       \(H_0: \mu = 0\) versus \(H_a: \mu \neq 0\) ,

  • Power is the probability of rejecting the null hypothesis when it is false

  • Ergo, power (as its name would suggest) is a good thing; you want more power

  • A type II error (a bad thing, as its name would suggest) is failing to reject the null hypothesis when it's false; the probability of a type II error is usually called \(\beta\)

  • Note Power \(= 1 - \beta\)

Note: Slide 3 and 4 have been based on the coursera lecture on power

Power (cont.)

  • Given the hypotheses on the previous slide, the power is
    \[P\left(\frac{\bar X - 0}{s /\sqrt{n}} > t_{1-\alpha/2,n-1} ~;~ \mu = \mu_a \right)\]
  • Note that this function depends on

   - the specific value of: \(\mu_a\);
   - the standard deviation: \(s\);
   - the number of subjects: \(n\);
   - the significance level: \(\alpha\).

  • It is also possible to calculate the number of subjects needed for a given power.

Slidify tool to calculate power

The slidify tool power calculation helps the researcher to calculate the number of subjects that are needed to get the desired power given the standardized difference and the desired power.

The following example caclulates the number of subjects needed for a standardized difference of 1, significance level of 0.05 and power of 0.8. 

nSubj<-ceiling(power.t.test(n= NULL, delta=1 ,sd=1,sig.level=0.05, power=0.8, 
                            type="paired",alternative="two.sided")$n)
print(nSubj)

## [1] 10
